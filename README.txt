Σε Linux, για να παραχθεί το εκτελέσιμο αρχείο, στο φάκελο /make-linux, από τερματικό τρέχουμε την εντολή:
		make
Για να καθαρίσουμε το build, στον ίδιο φάκελο, τρέχουμε:
		make clean

Χρήση:
		./proxyserver <port>
		όπου './' η εντολή εκτέλεσης και <port> ο αριθμός της θύρας που ακούει ο proxy server

Για να δείτε το αποτέλεσμα, αλλάξτε τις ρυθμίσεις proxy σας, μόνο για http,
ώστε να ακούει στο localhost, στη θύρα που διαλέξετε.
Έπειτα, επισκεφθείτε κάποιο site που να υποστηρίζει http και δείτε το αρχείο log.txt για λεπτομέριες.
Για να σταματήσετε τον server, πατήστε Ctrl + C ή Ctrl + Z. Μη ξεχάσετε να επαναφέρετε τις ρυθμίσεις proxy!
